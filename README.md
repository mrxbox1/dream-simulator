# "Dream Simulator"
Work in progress - the name pretty much explains it all

## How can I modify this project?
1. Install Godot 4.5.x at godotengine.org
2. Clone the project files (fork if needed)
3. Open the project.godot file in Godot (Import Existing Project)

## How can I compile this project?
See Godot 4.5's documentation at docs.godotengine.org/en/4.5/tutorials/export/index.html
